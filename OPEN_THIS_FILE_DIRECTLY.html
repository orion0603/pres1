Badir Blockchain Presentation - Windows Instructions

This guide provides the simplest and most reliable methods to run the Badir Blockchain Presentation on Windows.

Important: The presentation uses modern web technologies and requires Node.js to be installed on your computer.

Method 1: Run Vite Development Server Directly

  1. Open Command Prompt or PowerShell
  2. Navigate to the project folder:
    cd "path\to\your\project\folder"
  3. Then navigate to the client folder:
    cd client
  4. Install dependencies:
    npm install
  5. Run the Vite development server directly:
    npx vite
  6. Open your browser and go to:
    http://localhost:5173

Method 2: Using PowerShell (With Environment Variables)

  1. Open PowerShell
  2. Navigate to the project folder:
    cd "path\to\your\project\folder"
  3. Run this command to set up environment variables properly:
    $env:NODE_ENV="development"; npx tsx server/index.ts
  4. Open your browser and go to:
    http://localhost:5000

Method 3: Build and View Static Files

  1. Open Command Prompt
  2. Navigate to the project folder:
    cd "path\to\your\project\folder"
  3. Build the project:
    npm run build
  4. Install a simple HTTP server:
    npm install -g serve
  5. Serve the built files:
    serve -s client/dist
  6. Open your browser and go to the URL displayed in the terminal (usually http://localhost:3000)
Troubleshooting: If none of these methods work, you may try installing the project in a fresh directory or consider using a Linux-based environment like WSL (Windows Subsystem for Linux) for a more Unix-like experience.
